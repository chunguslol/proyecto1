Explicación de los Elementos Semánticos Utilizados:

  • <header>: Se usa para definir el encabezado de la página o una sección. Contiene información introductoria.
  • <nav>: Representa una sección de navegación, en este caso, dos veces: una para la navegación global y otra interna.
  • <main>: Es la parte principal del documento, donde se encuentra el contenido más relevante.
  • <article>: Define contenido autónomo dentro del documento, como un artículo o publicación.
  • <aside>: Representa contenido relacionado o adicional que complementa el contenido principal.
  • <footer>: Define el pie de página, donde se incluye información sobre derechos de autor.
